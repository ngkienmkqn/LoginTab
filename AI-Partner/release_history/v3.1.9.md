## Features

### Profile Exclusive Lock
**1 Profile = 1 User at a time** - Prevents session conflicts and anti-detection issues.

- Profile blocked if already in use by another user

- Time-based restrictions (5, 10, 30 minutes)

### Admin Kick Functionality
Admin/Super Admin can kick users from profiles with options:
- **No restriction** - Just kick, can reopen immediately
- **5/10/30 minute restriction** - Temporary ban
- **Revoke access** - Removes assignment entirely (can be reassigned later)
