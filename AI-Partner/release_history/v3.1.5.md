### Database Changes

```sql
-- New columns in accounts table
ALTER TABLE accounts ADD currently_used_by VARCHAR(36);
ALTER TABLE accounts ADD currently_used_by_name VARCHAR(255);

-- New table for usage history
CREATE TABLE profile_usage_log (
    id INT AUTO_INCREMENT PRIMARY KEY,
    account_id VARCHAR(36) NOT NULL,
    user_id VARCHAR(36) NOT NULL,
    username VARCHAR(255),
    action ENUM('open', 'close') NOT NULL,
    timestamp DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

### Performance Optimization

#### Hybrid-Only Sync Mode
- **DISABLED** heavy session folder backup (was 184MB+)
- **KEPT** lightweight cookies/localStorage sync (~KB)
- **KEPT** fingerprint sync in database
- Result: No more freezing on browser close, instant sync completion
